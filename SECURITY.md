# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.4.x   | :white_check_mark: |
| < 0.4.0 | :x:                |

## Reporting a Vulnerability

Report vulnerabilities by creating an issue: https://github.com/jclaggett/anascript/issues. I usually review these issues weekly.
